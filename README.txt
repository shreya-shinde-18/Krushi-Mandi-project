KRUSHI MANDI - HTML, CSS, JAVASCRIPT PROJECT
============================================

IMPORTANT: These files are for reference only and will NOT run in Figma Make. 
Copy these files to your local computer to use them.

HOW TO USE THESE FILES:
-----------------------

1. CREATE A NEW FOLDER on your computer called "krushi-mandi"

2. COPY ALL FILES from this project into that folder:
   - index.html
   - marketplace.html
   - sell-request.html
   - cart.html
   - login.html
   - payment.html
   - about.html
   - farmer-dashboard.html
   - buyer-dashboard.html
   - businessman-dashboard.html
   - admin-dashboard.html
   - styles.css
   - script.js

3. OPEN index.html in your web browser (Chrome, Firefox, Edge, etc.)

4. The website will work completely offline using LocalStorage for data

USER ROLES:
-----------
1. Admin - Full system access, view all users, orders, and requests
2. Farmer - Submit sell requests, compare mandi prices
3. Buyer - Browse marketplace, add to cart, place orders
4. Businessman - Review and accept/reject farmer sell requests

DATA STORAGE:
-------------
All data is stored in browser LocalStorage:
- users: Registered users
- sellRequests: Farmer sell requests
- orders: Buyer orders
- cart: Shopping cart items
- currentUser: Logged in user
